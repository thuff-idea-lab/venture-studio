## 1. Visual Direction
The overall look and feel of the Quote Tracker Template for Freelancers is clean, professional, and friendly. This style is essential for targeting freelance writers and consultants who value efficiency and organization in their work. A clean design minimizes distractions, allowing users to focus on their quotes, while a professional aesthetic builds trust and credibility. The friendly elements, such as rounded corners and soft colors, create an inviting atmosphere that encourages users to engage with the template.

## 4. Layout Pattern
The Quote Tracker Template is designed as a multi-page application with a simple navigation menu at the top. The maximum content width is 1200px to ensure readability. The spacing system uses a base unit of 8px, with common gaps of 16px and 24px for larger sections. The responsive approach is mobile-first, with breakpoints at 768px (tablet) and 1024px (desktop) to ensure an optimal experience across devices.

## 5. Component Library
- **Button**
  - Where it's used: Main actions (e.g., "Add Quote", "Save")
  - Customization notes: Rounded corners, primary color for main actions.
  
- **Input Field**
  - Where it's used: For entering quote details.
  - Customization notes: Light gray background, border color matching the border role.
  
- **Card**
  - Where it's used: To display individual quotes.
  - Customization notes: Light blue surface color with shadow for depth.
  
- **Table**
  - Where it's used: To list quotes and their statuses.
  - Customization notes: Alternate row colors for readability.

## 6. Key Screens
### Screen name and URL path
- **Dashboard** - `/dashboard`

### Layout description
The dashboard features a header at the top with the title "Quote Tracker". Below the header, there is a primary call-to-action button ("Add Quote"). The main content area displays a table of quotes, with a sidebar for navigation links to "All Quotes", "Pending Quotes", and "Completed Quotes".

### Key elements
- Header: "Quote Tracker" (h1)
- CTA Button: "Add Quote" (primary button)
- Table: Columns for "Quote", "Client", "Status", "Actions"
- Sidebar: Links to "All Quotes", "Pending Quotes", "Completed Quotes"
- Footer: Copyright notice

### Empty state
When no quotes have been added, the user sees a message: "No quotes yet. Click 'Add Quote' to get started!" with a friendly illustration.
