# Pre-Build Questions for Quote Tracker Template for Freelancers

Answer these questions before handing the spec to your coding agent. For each question, a suggested default is provided — accept it or override it.

## Branding & Identity

### What will be the name of the template?
**Why it matters:** The name is crucial for branding and marketing, as it should convey the purpose and appeal to the target audience.  
**Suggested default:** "Freelancer Quote Tracker"

### Do you want a custom domain for the template?
**Why it matters:** A custom domain can enhance credibility and make it easier for users to find the product online.  
**Suggested default:** Yes, use a domain like "freelancerquotetracker.com".

### Do you need a logo for the template?
**Why it matters:** A logo helps establish brand identity and can make the product more recognizable.  
**Suggested default:** Yes, create a simple logo that includes the product name.

### What tagline do you want to use?
**Why it matters:** A tagline can succinctly communicate the product's value proposition to potential users.  
**Suggested default:** "Effortlessly Track Your Quotes and Projects"

## Product Behavior

### What key features should the template include?
**Why it matters:** Defining core features ensures the template meets user needs and expectations.  
**Suggested default:** Basic fields for project name, client name, quote amount, status, and due date.

### Should the template include any pre-filled examples?
**Why it matters:** Pre-filled examples can help users understand how to use the template effectively from the start.  
**Suggested default:** Yes, include 2-3 sample quotes for demonstration.

### How should the template handle different currencies?
**Why it matters:** Freelancers may work with clients in various currencies, and this feature can enhance usability.  
**Suggested default:** Allow users to select their currency from a dropdown list.

## Monetization Setup

### Which payment method will you use for transactions?
**Why it matters:** Choosing a payment method is essential for processing sales and ensuring a smooth transaction experience for customers.  
**Suggested default:** Use Stripe for payment processing.

### Will you offer a money-back guarantee?
**Why it matters:** A money-back guarantee can increase customer trust and potentially boost sales.  
**Suggested default:** Yes, offer a 30-day money-back guarantee.

## Content & Data

### What initial copy do you want for the product description?
**Why it matters:** Clear and compelling product descriptions can significantly impact sales and user interest.  
**Suggested default:** "This customizable Excel template helps freelancers efficiently track and manage quotes for their projects, saving time and reducing errors."

### Do you want to include any instructional content or guides?
**Why it matters:** Providing guides can enhance user experience and help customers get the most out of the template.  
**Suggested default:** Yes, include a brief user guide as a separate document.

## Accounts & Services

### Which analytics service will you use to track sales and user engagement?
**Why it matters:** Analytics are crucial for understanding user behavior and improving marketing strategies.  
**Suggested default:** Use Google Analytics for tracking.

### Do you need an email service provider for customer communications?
**Why it matters:** An email service provider is essential for sending purchase confirmations, updates, and marketing communications.  
**Suggested default:** Use Mailchimp for email communications.
